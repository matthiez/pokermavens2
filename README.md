A simple PHP wrapper for Pokermavens2 from Briggs Softworks.

# Installation
1. Add to composer.json:  
```json
        "repositories": [
            {
                "type": "vcs",
                "url": "git@github.com:matthiez/pokermavens2.git"
            }
        ]
```
2. Run ```composer update``` in terminal.
